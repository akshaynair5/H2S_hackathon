Why It Matters

Misinformation affects daily decisions, from what we buy to what we believe. By putting AI-powered truth verification directly into your browser, TrustMeter builds a safer digital world where information is trustworthy and verifiable.

Why TrustMeter?

TrustMeter is a Chrome extension that detects misinformation on web pages in real time. It uses advanced AI and machine learning models to analyze text for fake or misleading information and identify AI-generated images. The extension provides a trust score, detailed explanations, and allows users to give feedback, helping build a community-driven fact-checking ecosystem that continuously improves accuracy and reliability.

Technical Overview

TrustMeter leverages a Chrome extension frontend with a Flask backend, integrating AI/ML models, vector databases, and external APIs for efficient misinformation detection.

Frontend Layer - Browser Extension

Lightweight Chrome extension that integrates seamlessly into your browsing experience.

Content Script Injector UI Overlay Manager Event Listeners Local Storage Cache API Client

AI/ML Processing Layer

Advanced machine learning models for text analysis, image verification, and credibility scoring.

Google Gemini API Google Fact Check API Google Cloud Vision API Image Detection Model Hugging Face Transformers (ML Classification) Google Custom Search API Sentence Transformers (Embeddings)

Data & Storage Layer

Hybrid storage with document database and vector search for caching and feedback.

Firestore - Articles & Feedback Normalized IDs & Embeddings Anonymous User Tracking Pinecone Vector Semantic Search TTL Expiration - 15 days

Data Flow

Step 1: Content Detection

Extension monitors webpage for text, images, and links that need verification.

Step 2: API Request

Detected content is sent securely to our backend services via encrypted channels.

Step 3: AI Analysis

Multiple AI models process content simultaneously for comprehensive evaluation.

Step 4: Score Calculation

Results are aggregated and a trust score is computed with supporting evidence.

Step 5: Result Delivery

Findings are sent back to the extension and displayed in user-friendly format.

Step 6: User Action

Users can view details, report issues, or proceed with informed confidence.

Get Started

Install TrustMeter in minutes and start browsing with confidence

Installation Guide

1

Download the Extension

Click the button below to download the latest version of TrustMeter.

2

Extract the ZIP File

Unzip the downloaded file. You'll find an /app folder containing all extension files.

3

Open Chrome Extensions

Navigate to chrome://extensions/ in your Chrome browser's address bar.

4

Enable Developer Mode

Toggle the Developer mode switch in the top-right corner of the extensions page.

5

Load the Extension

Click Load unpacked and select the /app folder you extracted earlier.

6

Start Verifying!

The TrustMeter icon will appear in your browser toolbar. Click it to activate verification on any webpage.

Quick Start Guide

1️⃣

Verify Text

Highlight any text on a webpage and right-click, then select "Verify with TrustMeter" to check for accuracy and bias.

2️⃣

Check Images

Right-click on any image and choose "Check Image Authenticity" to detect manipulation or AI generation.

3️⃣

Website Score

Click the TrustMeter icon in your toolbar to get an instant credibility score for the current website.

Troubleshooting

Extension Not Loading?

Ensure Developer mode is enabled in chrome://extensions/ and you've selected the correct /app folder.

Verification Not Working?

Check your internet connection and ensure the extension has necessary permissions. Try refreshing the webpage.

Slow Performance?

Clear your browser cache and disable other extensions temporarily to identify conflicts.
